React 前端開發學習成果展

本學期 React 學習重點回顧：

  • 進階 Hooks 實務：掌握 useRef 進行 DOM 控制，並透過 useMemo 與 useCallback 優化渲染效能。
  • 非同步與狀態防禦：運用 useState 與 useEffect 處理 API 資料請求，並實作條件渲染確保 UI 穩定性。
  • 專案架構思維：從元件化 (Component-based) 出發，將設計實作轉化為「資料驅動」的動態介面。
