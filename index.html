關於創作者 / About Me

大家好！我是目前就讀資訊工程與通訊相關科系的學生。本網頁為我的網頁程式設計期末專題成果。

我熱衷於軟體開發與演算法邏輯，這學期除了精進 C/C++ 程式設計與密碼學（如 Caesar/Vigenère 加解密實作）之外，更深入投入了全端網頁開發的領域。

個人技能棧： HTML5 / CSS3 / JavaScript / C++ / PostgreSQL 資料庫設計

本學期課程回顧 / Course Review

前端結構與排版

深入學習 HTML5 語意化標籤，並透過 CSS3 Flexbox 與 Grid 佈局建構具備響應式設計（RWD）的現代網頁介面。

異步 JavaScript 互動

掌握 DOM 操作、事件監聽（Event Listening），以及利用 Fetch/Async/Await 進行前後端 API 的資料傳輸。

雲端資料庫整合

學習關聯式資料庫（Relational Database）設計、CRUD 核心操作，並利用 BaaS 平台實作生產環境的資料部署。

期末專題介紹 / Final Project

專題名稱：校園匿名即時互動論壇系統

開發動機： 為了將課堂學到的靜態網頁美學與動態資料庫進行高難度整合。本專題旨在模擬 Dcard 與 PTT，打造一個高流暢度、零延遲、免登入的校園心聲交流平台。

核心技術： 前端 Vanilla JS 渲染 + 後端雲端資料庫 Supabase + 安全 RLS 存取政策 + GitHub Pages 靜態託管。
